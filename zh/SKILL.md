不是你不懂球。是这游戏的规则,本来就设计成让你慢慢输。

庄家的优势不叫运气,叫**水位**——每一场,它先从盘口抽走 6% 到 12%,世界杯更狠。你还没下注,钱已经少了一截。这不是阴谋论,是写死在规则里的数学。十年、一百万张彩票,这台机器一分不差地碾过每一个散户。

更阴的是:**这刀不是平着切的。** 它专门加厚在你最爱买的地方——"强队肯定赢"的短赔热门。你越觉得稳,它宰得越狠。体彩太清楚你会买德国了,于是把德国的价压到骨头里,你还以为捡了便宜,笑着把钱递过去。

这些年,散户就是体彩的饲料。真实的概率、真实的赔率,聪明钱手里一直攥着,你从来没份——你永远是后知后觉、接最后一棒的那个。

**这个项目,就是照向那张牌桌的灯。** 开源、免费。它只干一件事:把那些百万真金堆出来的"真实概率"直接糊到你脸上,让你头一回看清——这一注,庄家到底宰了你几个点;你那张引以为傲的串关,又是怎么一层层把钱喂进它嘴里的。

我们不承诺你暴富(往下看,会讲清楚为什么没人能)。我们只承诺一件事:**让你不再当那个冤大头。把交给庄家的智商税,砍到全网最低。一起。**

### 凭什么之一:聪明钱真的准——我扒了 50 场

口说无凭。本届世界杯每踢完一场,我就扒下 Polymarket 在**开赛前 60 分钟**那一刻的真实成交价(几百万真金当场押出来的概率),拿去对**真实比分**。50 场,150 条腿。

![校准图:市场说几成,就真的来几成](../assets/calibration.svg)

市场说"24% 会发生",真来了 23%;说"58%",兑现 65%——**几乎全趴在完美线上**。Brier 分 0.527(瞎蒙是 0.667),剔掉平局命中率 89%。这就是你做梦想要的"真实胜率"。
*(丑话:这是 Polymarket 一家、50 场,样本不大、薄桶有噪声,图上都标了。Pinnacle 和交易所没单独回测——但临场四家咬合在 1.5 个点内,唱同一个调。)*

### 凭什么之二:水位是藏起来的,而我们能看见

同一场**日本 vs 瑞典**:你买日本胜,EV **−18%**;买那条没人要的平,EV **几乎是 0**。一个是血亏,一个几乎不要钱——天壤之别,全藏在你看不见的地方。这套模型把每一注的真实成本扒光,告诉你水薄在哪——薄的下,厚的滚。

### 凭什么之三:串"热门"是陷阱,但串关本身不是原罪

串关感觉像博大的,代价是把水位**相乘**。同样几场球串**厚水位的热门**:2 串 1 预期亏 **−22%**;拆成薄水位单关只亏 **−1.5%**。庄家最爱凌晨攥着短赔热门串关的那个人——别当那个人。

**但要分清:** 串*薄腿*、为搏一个单关够不到的大上限,是另一回事——两条薄腿串出的总水位,可能比一条被狠抽的长赔单关还低。规矩是:**只串薄腿、腿数≤3、能先后下注就用 ride 别串。** 详见下方。

### 最后,丑话说尽:你赢不了庄家

长期你赢不了,没有人能,任何跟你说"稳赚"的都在骗你。这套模型不让你赢钱——它让你**输得最少、把下限焊死、偶尔偷它一个走运的夜晚**。只下你输得起的钱。看清水位。串关只串薄腿、绝不串热门。

这,才是我们的 **Beat the house**:不是把它打趴下——你做不到;是**再也不跪着,给它送钱**。

## 这是什么

一套给**体彩竞彩足球**下注用的纪律引擎。整套哲学一句话:

> **体彩是高水位的"软盘",锐利市场才是真值。我们专门狩猎体彩定价"最不离谱"的那几个点,在硬上限内下注,绝不碰体彩宰你最狠的地方。**

要锐利、要具体,更要**诚实**——这是真钱。

## 诚实条款 —— 每次都要讲清楚

**不要兜售幻觉。** 这个模型**长期赢不了庄家**——体彩的水位(每场约 6–12%,世界杯更重)是一堵墙,几乎没有散户能翻过去。它真正做的是:

1. **把你交给庄家的成本压到最低**(下薄水位的线,绝不下 −16% 那种宰客线)。
2. **按目标把上限的形状做到最好**——对的盈亏比×胜率、对的打法(单关 / ride / 串关),取决于你到底想干嘛(少亏 vs 搏翻倍 vs 博大上限)。
3. **把下限绝对焊死**(硬本金上限 = 娱乐成本)。

每份输出都按"**降低伤害 + 水位纪律**"来讲,不是"印钞机"。

## 驱动一切的两个数

对任意一个可下的选项(主胜 / 平 / 客胜):

```
真值概率  p  = 锐利市场算出的该结果发生概率
体彩赔率  o  = 体彩在该选项上给的小数赔率
EV (每1元) = p × o − 1     →  >0 下注;  =0 临界;  <0 亏
```

## 水位**不是平摊的**——这是核心edge

一场比赛的总水位(`Σ 1/赔率 − 1`)会被**堆在散户扎堆买的那一侧**——通常是短赔大热门,因为散户都压"强队肯定赢",体彩就把热门的赔率压得最狠。

结果:**同一场**比赛里,热门那条线可能是 −16% EV,而平局只有 −2%。**所以押水位最薄的那个选项,不管它是什么。**

⚠️ **不永远是平局!** 在势均力敌的比赛里,平局本身可能是最差的线(实测挪威–法国:平 −15.8%、冷门挪威 −9.8%,那场该押冷门)。**绝不臆测,每场都重算。**

## 值博线:盈亏比 × 胜率(衡量"值不值得博")

EV 只是把 p 和 o 揉成一个数。换个等价、更直观的视角:
- **盈亏比 b = 赔率 − 1**;**胜率 = p**。
- **值得博 ⟺ 胜率×赔率 `p·o > 1` ⟺ 盈亏比 `b > (1−p)/p` ⟺ `p > 1/o`**(三者同一件事)。
- **值博线缺口 = `b − (1−p)/p`**(<0 = 在线下方 = 体彩在抢你)。
- **低胜率也能值得博**——只要盈亏比够大过线(例:胜率 17% 配盈亏比 >4.9)。别只因胜率低就否决。
- 凯利下注比例 `f* = EV / 盈亏比`(≤0 一律别下)。

## 水位不是目标,但绝不能无视

水位(=EV)**不是目标函数**——体彩全场负EV,按水位排座次会让你永不出手。但它**不是没意义**:它是同一档奖里挑票的唯一标准。

```
P(达标) = (1 / 总赔率) × (1 − 水位)
```

- **盈亏比/总赔率**定"奖多大";**水位**定"为这个奖,你拿到几成公平概率"。
- 同样大的奖,水位越高命中越低——体彩把**最厚的水位专藏在最大盈亏比的票上**(长赔陷阱)。"盈亏比够大就无视水位"正是它设的陷阱。
- **排序键就用 `P(达标)`**:它一个数同时吃进盈亏比和水位。盈亏比挑奖档,水位挑那档里最划算的票。

## 目标决定打法(不预设、不偏袒任何一种)

先定目标,再选打法:
- **目标A · 少亏(风险中性)**:按 `EV` 或 `EV − λ·SD`(λ≈0.4)排,买最薄水位 1-2 条,单关。模式A 纯正EV在体彩几乎=不出手——这是对的、不是没逻辑。
- **目标B · 搏达标(翻倍 / 博大上限)**:负EV+有目标线的 **bold play**——集中、大胆、按 `P(达标)` 排,**不要平分摊薄**(平分是负EV下最差的达标策略)。"达标"可设翻倍下限,且上不封顶。
- 两个目标下水位都仍要看(它已进 P(达标)),只是不再是排序的唯一轴。

## 工作流

1. **时间闸先过**:体彩白天/晚上售票、卖到赛前~15-30分;世界杯多为**凌晨开赛**→必须白天**提前一次买好**→**ride 对凌晨球失效**(没法凌晨下第二关),能提前组合多场的只有**串关**。先剔除你买不到的场。
2. **拿体彩赔率(`o`)**——整版胜/平/负。**优先用你自己的体彩 App 或彩票照片**(精确)。爬取只是近似,见 `references/data-sources.zh.md`。
3. **拿锐利真值(`p`)**——去水(`1/赔率` 归一化到和为1):**Polymarket**(实时,回测 64.8%→66%)、**Betfair 交易所** + **PS3838/BetInAsia**(= Pinnacle 亚洲盘)、或 **oddsportal 共识**(较弱)。每个数标来源+时间+单源/交叉。
4. **枚举路径、按目标排**:`python scripts/ev_predator.py m.json --goal reach --target 2 --sigma 0.01`。单关 / ride(仅先后开赛) / 2-3串,按 `P(达标)` 与水位排前沿。
5. **定注额**:硬上限=你愿意全亏的钱,亏完不补。搏达标别平分、按 P(达标) 集中;同时报 P(达标) 和 P(全损);要"上限+少亏"用**杠铃**(主力薄腿注 + 一小笔独立高赔飞票)。
6. **出下注单 + 强制对照**(模板见下)。

### 铁律(改版,每次讲清为什么)
- **串关不再一刀切禁**:串**厚水位热门**是陷阱;串**薄腿**搏单关够不到的上限是另一回事。规矩:① 只串薄腿(每条接近值博线);② 腿数≤3(水位复利);③ 能 ride 就别串;④ 同时开赛、或单关够不到目标→才上串关。
- **硬上限 = 你愿意全亏的钱**,亏完不补——唯一真正的"亏损最小化"。负EV下凯利=0,这是娱乐预算不是投资仓。
- **水位不是目标,但绝不能无视**:按 `P(达标)` 排(已含盈亏比+水位);搏达标保持集中(Dubins–Savage"大胆下注"),别摊薄到归零。
- **每次必给「稳 vs 冲」对照**:最高 P(达标) 的稳注 vs 博上限注 + 为冲上限放弃多少命中率——即便用户只要一个结论也要摆,让他看见代价再拍板。一并报"全损概率"。

## 时机
- 用你**下注那一刻**的锐利价。黄金点 ~开赛前60分(阵容公布后);但**凌晨球这个窗口够不到**,只能白天提前买、接受真值是几小时前的快照(实测半天的漂移多在 0-3pt,主要风险是错过临场阵容那一跳)。
- 体彩**开赛前约 15–30 分钟停售**——留缓冲下单。
- 竞彩胜平负只按 **90 分钟**结算(不含加时点球)。淘汰赛拖进加时/点球的,仍按**平**结算——淘汰赛 90 分钟平局很常见。

## 数据来源——不可商量
**每个概率/赔率数字都必须带:(1)来源、(2)抓取时间、(3)单源还是多源交叉。** 绝不报裸概率。区分"锐利市场隐含概率(最佳代理)"和"真值(没人有)"。标注过期快照。

## 引擎
`scripts/ev_predator.py` 三种模式:
- `--goal scan`(默认):逐场 7 列(庄家胜率/赔率/真值/锐利/盈亏比/值博线缺口/EV)+ 每场最薄水位;`--sigma 0.01` 加稳健安全垫(高赔腿要求更厚)。
- `--goal reach --target 2`:枚举 单关/2串/3串,按 `P(达标)` 排前沿,**末尾强制打印「稳 vs 冲」对照**(`--pfloor` 调博上限注的命中率地板)。这是"今天怎么买"的主力。
- `--goal double --target 2`:满注前沿 + 打底×滚利润(ride)。
- `--bankroll N` + JSON 给 `bets`:枚举 2ⁿ 盈亏分布(P(达标)、P(全损)、最好/最坏)。

JSON 每场可带 `sharp_prob`(锐利列)、`sigma`、`kickoff`。跑 `--help` 看格式。优先用引擎,别手算。**诚实 > 让用户高兴。**
